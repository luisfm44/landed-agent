# DT 770 Pro Notes

The DT 770 Pro is often considered a strong studio headphone for monitoring and tracking.

## Practical notes

- It is commonly used in studios because it isolates well.
- It is durable and comfortable for long sessions.
- It can be useful for producers who need isolation.
- It is not always the first recommendation for final mix decisions if the user can use open-back headphones.

## Buying risk

When importing, check:
- impedance version
- condition: new, used, refurbished
- seller reputation
- shipping cost
- tax estimate
- warranty coverage
- return policy
